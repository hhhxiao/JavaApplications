## DigitalRecognition

### 简单说明

这是我大二上学期选秀*深度学习导论*课程做的大作业，可以实现手写数字（单个）的识别。安卓app只有简单的计算工作，用的都是训练好的数据。数据是用`matlab`训练出来的，整个过程涉及到矩阵卷积，后向传播，量化，池化等算法。初始数据集是被玩烂了的`minist`。最后成绩还是拿了优秀的(95+)。

### 代码说明

`code-matlab`文件夹是训练的代码，`code-matlab`是安卓app的主要代码（时间原因找不到完整的工程目录了）
