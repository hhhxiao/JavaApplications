## 说明

做完这个就打算javaweb了，但是自己的javaSE的路还没结束（买的两本java核心技术还没看完），初学者项目结构还比较乱，下一次的实战肯定是由结构图的。
项目来源是[这里](http://how2j.cn/k/hutubill/hutubill-tutorials/710.html)
